# ChIP-seq, First Take

My first real encounter with epigenomics. ChIP-seq had been a term I knew existed, this was the first time I actually worked with the data.

This is a course practical from an Integrative Biology course at the University of Padova. The analysis covers ChIP-seq data from Merkel cell polyomavirus transcription factors (ST-HA, EP400 and MAX) using the GSE69877 dataset. Peak profiling, genomic annotation, cross-TF pathway comparison with KEGG, peak overlap testing, and Reactome enrichment on genes shared across all three factors.

Epigenomics is a direction I want to go deeper into one day.
